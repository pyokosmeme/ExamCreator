Trigonometry & Calculus

Exam #3 Key Equations

Capacitance
Parallel-Plate Capacitor
C = ε₀ A/d
With dielectric: C = κε₀ A/d
Spherical Capacitor
C = 4πε₀(ab)/(b-a)
With dielectric: C = 4πκε₀(ab)/(b-a)
Cylindrical Capacitor
C = 2πε₀L/ln(b/a)
With dielectric: C = 2πκε₀L/ln(b/a)
Capacitors in Series
1/Ceq = 1/C₁ + 1/C₂ + 1/C₃ + ...
Capacitors in Parallel
Ceq = C₁ + C₂ + C₃ + ...
Energy Stored in a Capacitor
U = ½QV = ½CV² = ½Q²/C
Energy Density in Electric Field
u = ½ε₀E²
With dielectric: u = ½κε₀E²
Inside linear dielectric: E = E₀/κ (for the same free charge configuration)
Variable definitions
  • C: capacitance (farads, F)
  • κ: dielectric constant (relative permittivity, unitless)
  • A: plate area (m²)
  • d: plate separation (m)
  • a, b: inner and outer radii (m)
  • L: cylinder length (m)
  • V: potential difference (volts, V)
  • U: stored energy (joules, J)
  • u: energy density (J/m³)
Note: constants (e, ε₀) are listed in Useful constants.
Current and Resistance
Current and Current Density
I = dQ/dt
J = nevd
I = ∫J·dA
Drift Velocity
vd = I/(neA)
n = number density of carriers
Resistance and Resistivity
R = ρL/A
ρ = E/J = m/(ne²τ)
Temperature Dependence
R = R₀[1 + α(T - T₀)]
ρ = ρ₀[1 + α(T - T₀)]
Ohm's Law
V = IR
J = σE
σ = 1/ρ
Variable definitions
  • I: current (amperes, A)
  • Q: charge (coulombs, C)
  • t: time (s)
  • J: current density (A/m²)
  • n: charge carrier density (1/m³)
  • v_d: drift velocity (m/s)
  • A: cross‑sectional area (m²)
  • ρ: resistivity (Ω·m)
  • σ: conductivity (S/m), σ = 1/ρ
  • m: carrier mass (kg)
  • τ: mean free time between collisions (s)
  • α: temperature coefficient (1/K)
  • R: resistance (ohms, Ω)
  • L: conductor length (m)
  • V: voltage (volts, V)
Note: constants (e, ε₀) are listed in Useful constants.
DC Circuits
Kirchhoff's Rules
Junction rule: ∑Iin = ∑Iout
Loop rule: ∑V = 0
Resistors in Series
Req = R₁ + R₂ + R₃ + ...
Resistors in Parallel
1/Req = 1/R₁ + 1/R₂ + 1/R₃ + ...
Voltage Divider
Vout = (Rout/(R₁ + R₂)) × Vin
Current Divider
I₁ = (R₂/(R₁ + R₂)) × Itotal
Power
P = IV = I²R = V²/R
EMF and Terminal Voltage
Vterminal = ε - Ir
r = internal resistance of source
Variable definitions
  • ε: emf (volts, V)
  • r: internal resistance of source (Ω)
  • R_eq: equivalent resistance (Ω)
  • R₁, R₂, …: individual resistances (Ω)
  • I_total, I₁: total and branch currents (A)
  • V_in, V_out: input/output voltages (V)
  • P: power (watts, W)
Note: constants (e, ε₀) are listed in Useful constants.
RC Circuits
Time Constant
τ = RC
Charging a Capacitor
q = C·ε(1 - e-t/τ)
VC = ε(1 - e-t/τ)
I = (ε/R)e-t/τ
Discharging a Capacitor
q = q₀e-t/τ
VC = V₀e-t/τ
I = (V₀/R)e-t/τ
Energy in RC Circuit
U = U₀e-2t/τ
Energy dissipated: U₀(1 - e-2t/τ)
Variable definitions
  • R: resistance (Ω)
  • C: capacitance (F)
  • τ: time constant = RC (s)
  • ε: emf / source voltage (V)
  • q, q₀: capacitor charge (C)
  • V_C, V₀: capacitor voltage (V)
  • I: current (A)
  • t: time (s)
  • U, U₀: energy stored in capacitor (J)
Note: constants (e, ε₀) are listed in Useful constants.
